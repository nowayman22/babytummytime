♡ Baby Tummy Time

Arch Linux pacman repository — install with a single command.

Install

1. Add the repo to /etc/pacman.conf:

[babytummytime]
Server = https://nowayman22.github.io/babytummytime/repo

2. Sync and install:

sudo pacman -Sy babytummytime
