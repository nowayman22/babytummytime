🐧 Arch Linux

Add the repository to /etc/pacman.conf:

[babytummytime]
SigLevel = Optional TrustAll
Server = https://nowayman22.github.io/babytummytime/repo

Then sync and install:

sudo pacman -Syu babytummytime
